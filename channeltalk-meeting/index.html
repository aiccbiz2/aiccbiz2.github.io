채널톡 상품 등록 — LG 유플러스 플랫폼 연동

채널톡을 LG 유플러스 클라우드 고객센터 상품 라인업에 등록하는 방안 논의. 기존 브리지텍, 제네시스 클라우드, GS네오텍, 유비커스 등과 유사한 마진 쉐어 구조 검토.
  • 상품 등록 vs. 직접 제안(PoC) 방식 병행 고려
  • SME · 미드마켓 타겟은 상품 등록, 엔터프라이즈 타겟은 직접 제안·RS 구조
  • 카페24·메이크샵 등 웹호스팅 빌더 자동 정산 구조 참고
상품 등록 마진 쉐어 SaaS/클라우드

STT 연동 파트너십 — ECSTT 기술 협력

LG 유플러스의 자체 STT(ECSTT)를 채널톡에 연동하는 기술 파트너십 방안. 통신사 고객센터 기반 모델로 인식률 92~97% 수준.
  • 베이스 모델 인식률 92%, 튜닝 후 95~97%
  • 금융권 특화 학습 완료, 도메인 확장 가능
  • 사용량 기반 과금이 아닌 AWS 내 구축 → 고정 비용 구조로 비용 경쟁력 확보
  • 채널톡 측 현재 OpenAI STT 사용 중
STT 연동 ECSTT 기술 협력

타겟 고객사 영업 전략 — 공동 PoC 추진

LG 유플러스 기존 고객DB와 채널톡 성공 사례를 결합한 공동 영업 전략 수립.
  • 스위트스팟: 30~200석 규모, 리테일·제조 버티컬
  • 우선 제외: 금융·보험·공공기관 (온프레미스 이슈)
  • 주목 버티컬: 외국계 기업 (렉서스 코리아, 르노 코리아 등), 라스트마일·유통
  • 일본 루트: 유플러스 일본 지사·후지츠 출신 임원 활용
  • 3~4개 공동 타겟 선정 후 직접 어프로치
공동 영업 외국계 기업 리테일/제조

에스원(S1) 무인 매장 관제 — 즉시 실행 기회

전국 약 1만 개 무인 매장 관제 시스템을 채널톡 기반으로 구축하는 구체적 기회 발굴. REST API 기반 IVR 연동 방식으로 즉시 구현 가능.
  • 무인 매장 키오스크 ↔ 비밀번호 해제·OTP 자동화
  • 구독형 클라우드 구조 요구 (인프라 없이 가능)
  • CRM 인증값 연동만으로 구현 가능
  • 미팅 당일 내부 영업팀에 즉시 공유 결정
에스원(S1) 즉시 실행 무인 매장

옴니채널 AX 컨설팅 — 콜+채팅 통합 전략

LG 유플러스의 콜(IPCC) 기반 AX 분석과 채널톡의 채팅 AX를 결합한 옴니채널 컨설팅 패키지 구성.
  • 콜 분석(PI) + 채팅 분석을 합치면 "사기급" 옴니채널 솔루션
  • AX 팀: 고객 로우데이터 수집 → 현황 분석 → 시나리오 설계 → 솔루션 제안
  • 채널톡 API ↔ 구글시트 연동 → 통계 레포트 자동화 가능
  • 아마존 커넥트 연동으로 대기업 타겟 상품 구성 검토
옴니채널 AX 컨설팅 Amazon Connect

핵심 결론 요약

LG 유플러스와 채널톡은 단순 상품 등록 방식보다 직접 PoC 제안 + RS(Revenue Share) 구조를 우선 추진하기로 합의했다.

협력의 핵심은 ①채널톡 영업망 + LG 유플러스 고객DB 결합, ②STT 연동으로 옴니채널 AX 컨설팅 패키지화, ③에스원 무인 매장 관제를 즉시 실행 기회로 삼는 것이다.

양사 모두 기존 카르텔(BT, 제네시스, 시스코 등) 외부에서 새로운 시장을 개척하겠다는 의지가 일치했으며, "판은 저희가 만들 테니 PI가 기대된다"는 발언처럼 강한 실행 의지가 확인됐다.
